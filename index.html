HTML

Voltar

A tag principal de uma página HTML, é ela que envolve todas as outras tags da página e torna possível a construção dos blocos essenciais para sua montagem.

head

Voltar

O head também é uma das tags principais do HTML. Ela abriga a tag title, que muda o título da página, e a tag style, que faz mudanças no CSS.

body

Voltar

A tag body abriga a maioria das tags da página HTML. É nela que estarão as outras tags com imagens, textos, tabelas e muito mais que usamos para compor sites e programas.

title

Voltar

O title dá o título à página HTML. É o seu conteúdo que aparece na aba do navegador.

b

Voltar

A tag b deixa o texto em negrito, assim como a tag strong, porém possui uma relevância semântica menor, principalmente para leitores acessíveis de páginas web.

strong

Voltar

Muito parecida com a tag b, a tag strong torna o texto negrito. Comparada à tag b, a tag strong possui uma relevância semântica maior e dá mais destaque ao texto quando lido por um programa de leitura acessível de páginas web.

u

Voltar

A tag u torna o texto sublinhado. A letra que dá nome à tag é a inicial da palavra inglesa underline, ou linha inferior (sublinhado). É usada para destacar termos.

i

Voltar

A tag i torna itálicas as palavras envolvidas por ela. Este é um recurso muito utilizado para destacar palavras estrangeiras.

h1, h2, h3, h4, h5 e h6

Voltar

Estas são tags para títulos textuais, sendo h1 a maior tag e h6, a menor. O título h1, h2, h3, h4, h5 e h6, por exemplo, foi escrita com a tag h2. Para textos menores e longos constuma-se usar outra tag, a p.

p

Voltar

A tag p abre um parágrafo. É reservada para textos longos e com fonte de tamanho menor que os títulos. Aqui é interessante utilizar recursos como links, negritos, itálicos e muito mais.

br

Voltar

Esta é uma das tags mais utilizadas em páginas HTML. br é basicamente utilizada para fazer quebras de linha e fazer com que o conteúdo a seguir seja exibido na linha de baixo. Tem origem na palavra quebra, do inglês break.

blockquote

Voltar

Blockquote é uma tag que formata o texto em formato de citação, com maior margem que o texto da tag p.

ul

Voltar

Esta tag cria uma listagem não ordenada. Com ul, os itens (criados com a tag li) são precedidos de um bullet point e devem ser criados entre a abertura e o fechamento da tag ul.

ol

Voltar

A tag ol cria uma lista ordenada, com os itens iniciados por números iniciando por 1. Estes itens devem ser inseridos entre a abertura e o fechamento da tag ol, com a tag li.

li

Voltar

li é a tag que cria itens a serem inseridos em listas, sejam elas não ordenadas ou ordenadas.

a

Voltar

Uma tag muito utilizada atualmente, a é utilizada para criar links. O link é fornecido a partir do atributo href e tem o comportamento definido pelo atributo target, com as opções de abertura do endereço na mesma ou em páginas novas.

img

Voltar

Similar à tag figure, a tag img faz a inserção de uma imagem à página HTML. Ela aceita os atributos src, que traz a origem da image, height, que define a altura e width, que define a largura. Apenas informar um dos valores faz com que a imagem seja redimensionada proporcionalmente.

audio

Voltar

Utilizada especificamente para exibir conteúdos de áudio em páginas HTML, a tag audio possui atributos como autoplay, que decide se a áudio vai ser iniciado automaticamente com a abertura da página, e controls, que decide se os controles de play, audio ou download vão ser exibidos para o visitante do site.

video

Voltar

De forma semelhante, à tag audio, a tag video insere conteúdo multimídia numa página HTML, desta vez em formato de vídeo. Também é preferencial em detrimento da tag embed por ter maior valor semântico. Alguns de seus atributos são source, loop e autoplay.

track

Voltar

track é uma tag específica para a adição de legendas para vídeos ou áudios, com opção de inserir arquivos de mais de uma língua para serem selecionados.

form

Voltar

Muitas das inserções de informações são feitas a partir de formulários. E é para isso que serve a tag form. No formulário são inseridos campos de entrada de informações como inputs.

input

Voltar

Os campos de texto para inserção de informações dentro entre tags form são criados por meio das tags input, sendo elas com o tipo definido pelo atributo type, com as opções de ser text, ou texto comum, number, ou números, button, ou botão, entre outras possibilidades.

button

Voltar

A tag button cria um botão clicável, com um dos principais atributos sendo o type, com as opções button, sendo um botão clássico, reset para limpar um formulário ou submit, para enviar os campos de um formulário.

select

Voltar

select cria uma lista expansível para seleção de opções, sendo utilizada em conjunto com as tags label para nomear o campo e form para envolver inputs e selects.

label

Voltar

A tag label exibe rótulos para campos de formulário como inputs ou selects.

textarea

Voltar

Esta tag é utilizada para fazer uma inserção de texto em um formulário, tendo mais exibição de linhas que o input.

font

Voltar

A tag font é utilizada para manipular a fonte dentro de um bloco de texto. Se formos, por exemplo, mudar a fonte de uma palavra específica, pode-se envolvê-la com a tag font e manipular seu CSS.

div

Voltar

A tag div é muito utilizada no HTML por criar blocos estruturais que abrigam diversas outras tags. Possui comportamento de block, ocupando toda a largura da página, diferente da tag span.

span

Voltar

A tag span tem comportamento semelhante à da div, mas enquanto aquela apresenta o atributo display como block, esta apresenta como inline e permite ocupar apenas a largura do seu conteúdo.

fieldset

Voltar

fieldset é a tag que agrupa campos em um formulário. Ela pode ser utilizada para separar questões sobre dados pessoais das questões socioeconômicas num questionário sobre alunos, por exemplo. É utilizada em conjunto com a tag legend, que cria legendas para nomear os agrupamentos feitos pela fieldset.

legend

Voltar

Utilizada dentro de um fieldset, a tag legend cria um rótulo superior para nomear esses blocos de informações.

embed

Voltar

A tag embed é utilizada para trazer um conteúdo externo para nosso site. Pode ser uma página HTML, um vídeo ou uma imagem. Porém, tem menor valor semântico que outras tags, como a img, video ou iframe, por isso não é mais recomendada.

iframe

Voltar

iframe é uma tag utilizada para trazer conteúdos externos para nosso site, tais como vídeos do Youtube ou Google Maps. Em nosso código, o conteúdo pode até mesmo se comportar como se houvesse outra tag HTML dentro da tag embed.

table

Voltar

A tag table é responsável por criar tabelas numa página HTML. Ela é utilizada em conjunto com diversas outras tags pertinentes à construção de tabelas, como tr, th e td.

tr

Voltar

Sigla de table row, ou linha de tabela, a tag tr abriga informações que vão compor uma linha dentro de uma tabela, podendo ser um cabeçalho ou uma linha comum de conteúdo.

td

Voltar

O conteúdo inserido dentro da tag td vai ser exibido em uma célula de tabela. Cada linha pode conter inúmeras células paralelas e abrir texto, links ou imagens.

th

Voltar

Similar à td, a tag th faz a inserção de uma informação em uma célula de uma tabela. Porém, neste caso, a informação vai ser considerada um cabeçalho e, por isso, geralmente vai ser utilizada na primeira linha da tabela.

tbody

Voltar

Uma forma de melhorar a semântica de um código é separar as linhas de uma tabela em porções menores, sendo elas o cabeçalho, corpo e rodapé de tabela. A tag tbody entra neste contexto como o corpo da tabela, ou o lugar em que a maioria das informações vai ser inserida.

thead

Voltar

De forma semelhante à tbody, a tag thead é feita para separar o conteúdo de uma tabela de uma forma mais semântica. Esta tag é utilizada para envolver as linhas de cabeçalho de uma tabela, ou seja, a th.

tfoot

Voltar

Da mesma forma que as tags thead e tbody, a tfoot é utilizada para separar o conteúdo de uma tabela de forma mais semântica. Neste caso, todo o conteúdo de rodapé de tabela seria inserido na tfoot.

main

Voltar

A tag main é utilizada para aumentar a semântica das informações de uma página HTML, separando o conteúdo em cabeçalho, corpo e rodapé. O corpo, ou informações mais relevantes, é inserido dentro da tag main, enquanto o cabeçalho fica na tag header e o rodapé, na footer.

header

Voltar

header é a tag que envolve o conteúdo de "cabeçalho", dentro da tag body. É uma forma de dividir em três partes o conteúdo de corpo de uma página HTML, aumentando sua semântica.

footer

Voltar

footer é utilizada no contexto de dividir o conteúdo de corpo de uma página HTML em três páginas, header, main e footer. footer abriga o conteúdo final do corpo da página, o rodapé.

aside

Voltar

No contexto do aumento da semântica de uma página HTML, a tag aside é utilizada para inserir o conteúdo secundário com relação ao seu tema principal. A contraparte da aside é a section.

section

Voltar

A tag section abriga o conteúdo relacionado ao tema principal do site, dividindo-o em blocos ou seções. O conteúdo secundário deve ser inserido na tag aside.

nav

Voltar

nav é a tag que envolve informações pertinentes à navegação na página. É uma das ferramentas para utilizadas para aumentar a semântica na página HTML.

article

Voltar

A tag article tem a função de conter conteúdo, seja para notícias, artigos, posts em fórum ou em blogs.

q

Voltar

q é a tag feita para envolver uma citação: exemplo. Do inglês quote, o texto inserido nesta tag será exibido entre aspas.

figure

Voltar

Esta tag é utilizada para inserir imagens em uma página HTML. Diferente da tag img, a tag figure trabalha em conjunto com a tag figcaption.

figcaption

Voltar

A tag figcaption insere legendas em imagens inseridas com a tag figure. A legenda aparece abaixo da imagem.

picture

Voltar

Esta é uma mais uma tag para inserção de imagens em uma página HTML, mas diferente das tags img e figure, a tag picture permite subir versões em diferentes tamanhos para que sejam exibida a versão ideal para determinado tamanho de tela.

small

Voltar

A tag small é utilizada para reduzir o tamanho de uma porção do texto. Exemplo: palavra.

mark

Voltar

mark é tag utilizada para destacar uma porção do texto. A tag aceita os atributos background-color e color.

sub

Voltar

A tag sub é utilizada para fazer com que uma parte do texto fique deslocada para baixo com relação à linha. Sua contrapartida é a tag sup.

sup

Voltar

De forma oposta à tag sub, a tag sup desloca uma porção de texto para a parte superior da linha.

del

Voltar

A tag del torna uma porção de texto tachada. Ao ser utilizada em conjunto com a tag ins, que sublinha o texto de forma semelhante à tag u, passa a ideia de que uma parte de informação foi retirada e outra inserida em seu lugar.

abbr

Voltar

abbr é a tag usada para abreviações. Seu diferencial é que, ao pousar o ponteiro do mouse no seu conteúdo, uma versão completa da abreviação vai aparecer. Exemplo: CPF.
